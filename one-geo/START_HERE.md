# 🌍 Well-Log Data Analysis System

## 🚀 Quick Start (2 minutes)

### Automated Setup (Recommended)
```bash
# macOS/Linux
bash install.sh

# Windows
install.bat
```

### Manual Setup
```bash
# Terminal 1 - Backend
cd backend
npm install
npm run dev

# Terminal 2 - Frontend
cd frontend  
npm install
npm run dev

# Open: http://localhost:3000
```
